# js-mqtt

A simple MQTT Subscriber written in JS

## Prerequisites

1. Node.js > 10
2. npm > 6
3. MongoDB
4. ThingsBoard

## Usage

1. Clone this repository
2. Install all the dependencies using `npm install`
3. Run using `npm run dev`
